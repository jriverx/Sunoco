# Sunoco Auto Repair Prototype Website

Prototype website created for a **Sunoco-affiliated automotive repair shop in Virginia, USA**.  
The project was designed as a modern, conversion-focused web presence for a local workshop, aiming to improve customer trust, showcase services, and streamline appointment bookings.

> **Note:** This was a prototype/concept project and was never deployed into production.

---

## Overview

This project was built to provide a professional digital storefront for an automotive repair business operating under the Sunoco brand. The goal was to create a clean, trustworthy, and mobile-friendly website that could help generate more leads and improve customer communication.

The prototype focuses on:

- Presenting the shop’s repair and maintenance services
- Displaying customer trust elements (reviews, certifications, guarantees)
- Simplifying contact and booking flow
- Improving local SEO visibility
- Creating a stronger online brand presence
